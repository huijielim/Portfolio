Projects

This page features 3 projects of Eye, Pacman and RealTimeBusTracker. The webpage has been built using Bootstrap, with the homepage using https://getbootstrap.com/docs/5.0/examples/cover/ and the projects page using https://getbootstrap.com/docs/5.0/examples/album/

This project is setup to have 2 eyes on the web interface. When the mouse moves the eyes would follow the mouse movement. Try to move your mouse as fast as you can around the web interface. Try moving out of the web interface and see if the eyes still follow you.

This project is setup to initiate a pacman image on the web interface and to move the pacman on the web interface. Click "Add PacMan" to add more pacmen. Click "Start Game" to get the pacmen moving around the web interface. The pacmen would bounce off the walls of the web interface.

This project is setup with a map pre-setup on the web interface. A marker would be created to identify where the location of a bus is and as the bus moves throughout the map, the marker would follow the bus to indicate the location of the bus in real time.
